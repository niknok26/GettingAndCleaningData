# Coursera: Data Science Specialization
## Getting and Cleaning Data Course Project
The purpose of this project is to demonstrate your ability to collect, work with, and clean a data set.

* UCI HAR Dataset

  + features_info.txt
  + features.txt
  + README.txt
  + test(folder)
